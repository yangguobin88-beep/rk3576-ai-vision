## 📋 检测场景分类

### 🟢 第一层：基础通用检测（YOLOv8 预训练已支持）

这些 **不需要重新训练**，YOLOv8 预训练模型已经能识别：

| 类别 | 包含对象 |
|------|----------|
| 人员 | 人 (person) |
| 交通工具 | 汽车、卡车、公交车、摩托车、自行车 |
| 动物 | 狗、猫、鸟、马、牛、羊 |
| 常见物品 | 椅子、桌子、笔记本电脑、手机、杯子、瓶子 |

**共 80 个类别**（COCO 数据集）

## 🚀 推荐学习路径

### 阶段 A：用公开数据集练手（1-2 天）

**推荐：安全帽检测**
- 数据量适中（约 7000 张）
- 场景明确容易理解
- 有现成的公开数据集

### 阶段 B：尝试缺陷检测（2-3 天）

**推荐：钢材表面缺陷（NEU-DET）**
- 经典工业数据集
- 6 种缺陷类型
- 图片清晰标注规范

### 阶段 C：自定义场景（按需）

根据你的实际项目需求训练

## 📊 公开数据集推荐

### 1. 安全帽检测数据集 ⭐⭐⭐⭐⭐

| 属性 | 值 |
|------|------|
| 名称 | Safety Helmet Detection |
| 图片数 | ~7000 |
| 类别 | helmet, head, person |
| 来源 | Kaggle / Roboflow |
| 格式 | YOLO 格式 |

**下载地址：**
- Roboflow: https://universe.roboflow.com/search?q=safety+helmet
- Kaggle: https://www.kaggle.com/datasets/andrewmvd/hard-hat-detection

### 2. NEU 钢材表面缺陷数据集 ⭐⭐⭐⭐

| 属性 | 值 |
|------|------|
| 名称 | NEU Surface Defect Database |
| 图片数 | 1800 (300/类) |
| 类别 | crazing, inclusion, patches, pitted, rolled-in, scratches |
| 格式 | 需转换为 YOLO |

**下载地址：**
- http://faculty.neu.edu.cn/songkechen/zh_CN/zdylm/263270/list/

### 3. MVTec 工业缺陷数据集 ⭐⭐⭐⭐

| 属性 | 值 |
|------|------|
| 名称 | MVTec Anomaly Detection |
| 类型 | 异常检测（不是目标检测） |
| 场景 | 瓶子、电缆、胶囊、金属螺母等 |

### 4. PCB 缺陷检测数据集 ⭐⭐⭐

| 属性 | 值 |
|------|------|
| 名称 | PCB Defect Dataset |
| 图片数 | ~1500 |
| 类别 | missing_hole, mouse_bite, open_circuit, short, spur, spurious_copper |

## 🎯 建议：从安全帽检测开始

**理由：**
1. ✅ 数据量充足，效果好
2. ✅ 场景通用，很多工厂都需要
3. ✅ 容易理解和验证
4. ✅ 可以快速看到训练成果

**步骤：**
1. 下载安全帽数据集
2. 用 train_yolov8.py 训练
3. 部署到 RK3576
